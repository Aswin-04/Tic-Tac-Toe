# Tic-Tac-Toe

This is a simple implementation of the classic Tic-Tac-Toe game using HTML, CSS, SCSS, and JavaScript.

## Description

Tic-Tac-Toe is a two-player game where players take turns marking spaces in a 3Ã—3 grid. The player who succeeds in placing three of their marks in a horizontal, vertical, or diagonal row wins the game.

## Features

- Two-player game
- Interactive and responsive UI
- Highlights winning combination
